# REFERENCIA RÁPIDA - Guía de 2 Minutos

## ?? Lo Que Necesitas Hacer YA

### 1. CONFIGURAR CONEXIÓN (1 minuto)

Abre: **`DATOS\clsConexion.cs`** línea 7

Busca esto:
```csharp
private string cadenaConexion = "Server=SERVIDOR;Database=BASEDATOS;User Id=USUARIO;Password=CONTRASEÑA;";
```

Reemplaza con TU servidor, por ejemplo:
```csharp
// Si tienes SQL Server Express instalado localmente:
private string cadenaConexion = "Server=.\\SQLEXPRESS;Database=ProducBD;Integrated Security=true;";

// O si tienes credenciales:
private string cadenaConexion = "Server=localhost;Database=MiBaseDatos;User Id=sa;Password=MiPassword;";
```

## ?? Si Algo Falla

| Problema | Solución |
|----------|----------|
| Error de conexión | Verifica cadena de conexión |
| "Tabla no existe" | Ejecuta SCRIPT_BD.sql completo |
| "No hay datos" | Inserta datos con INSERT en SQL |
| Error al compilar | Verifica namespaces en using |
| DataGridView vacío | Verifica conexión y datos en BD |
